# Fraud Detection in Mobile Payment Systems

Code accompanying a co-authored book chapter comparing machine learning models for detecting fraudulent transactions in mobile payment systems.

📄 **Published chapter:** [Enhancing Fraud Detection in Mobile Payment Systems: A Comparative Analysis of Machine Learning Models](https://www.taylorfrancis.com/chapters/edit/10.1201/9781003589273-39/enhancing-fraud-detection-mobile-payment-systems-comparative-analysis-machine-learning-models-aditi-singh-shivantika-tripathi-sandeep-chaurasia-amit-kumar-gupta-dayananda) — Taylor & Francis
**Authors:** Aditi Singh, Shivantika Tripathi, Sandeep Chaurasia, Amit Kumar Gupta, Dayananda

## What this covers

- Comparative evaluation of multiple ML classifiers on mobile payment transaction data, to identify which models best distinguish fraudulent from legitimate transactions
- Data preprocessing and feature analysis for transaction-level fraud signals
- Model performance comparison across the candidate algorithms

## Contents

`fraud-Detection-in-Mobile-Payment-Systems.ipynb` — full analysis: data preparation, model training, and comparative evaluation.
